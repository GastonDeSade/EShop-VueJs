# EShop-VueJs

Ce projet est un site d'Ecommerce. Il est simple et ne sert qu'à démontrer ma capacité à faire un site en vue ainsi que de me servir de mon [API N-tier](https://github.com/GastonDeSade/N-Tier).

Le site contient :

- Une page d'accueil avec la liste des produits
- Une page produit
- Une page de connexion
- Une page d'inscription
- Une page pour le panier
- Une page checkout
- Une page listant les commandes de l'utilisateur

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
